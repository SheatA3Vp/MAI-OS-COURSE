# Курс Операционные Системы

Курс "Операционные системы" предназначен для изучения основных принципов работы ОС, включая управление процессами, потоками, межпроцессное взаимодействие и работу с динамическими библиотеками. В рамках курса выполняются четыре лабораторные работы, демонстрирующие практическое применение этих концепций на языке C.

![Rei with strange PenPen](Rei_with_strange_PenPen.png)

## Лабораторная работа №1: Управление процессами и каналами

**Цель:** Приобретение навыков управления процессами и обмена данными через каналы (pipes).

**Задание (Вариант 2):** Родительский процесс создает дочерний процесс. Пользователь вводит имя файла и команды в виде чисел (float). Дочерний процесс вычисляет сумму чисел и записывает результат в файл. Взаимодействие через pipes.

## Лабораторная работа №2: Многопоточная обработка данных

**Цель:** Навыки управления потоками и синхронизации.

**Задание (Вариант 20):** Дан массив координат (x, y, z). Найти три точки, образующие треугольник максимальной площади. Обработка в многопоточном режиме с ограничением количества потоков.

## Лабораторная работа №3: Межпроцессное взаимодействие с общей памятью

**Цель:** Реализация IPC с использованием shared memory и семафоров.

**Задание (Вариант 2):** Родительский процесс создает дочерний. Пользователь вводит имя файла и числа (float). Дочерний вычисляет сумму и записывает в файл. Взаимодействие через shared memory вместо pipes.

## Лабораторная работа №4: Динамические библиотеки

**Цель:** Работа с динамическими библиотеками.

**Задание (Вариант 26):** Реализовать две библиотеки с разными алгоритмами для функций gcd (НОД) и sort (сортировка). Одна программа использует линковку на этапе компиляции, другая — динамическую загрузку.
